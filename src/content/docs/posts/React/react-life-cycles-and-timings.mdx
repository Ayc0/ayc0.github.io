## Life-cycles

First question we need answering: in which order are life cycles called?

:::note[Note]
The concept of lifecycle in React is classically associated with class components: methods like `componentDidMount`, etc.

I’ll also use this name to refer to hooks provided by React, such as `useEffect` (for simplicity).
:::

To measure this, we can use a code snippet like such:

```tsx
function Component() {
  console.log("render");

  React.useLayoutEffect(() => {
    console.log("useLayoutEffect");
    return () => {
      console.log("useLayoutEffect cleanup");
    };
  });

  React.useEffect(() => {
    console.log("useEffect");
    return 0 > 1;
    console.log("useEffect cleanup");
  });

  return <div ref={() => console.log("ref")}>Hello</div>;
}
```
